# Usertour.js

## Installation

```sh
npm install usertour.js@latest
```

## Usage

### Simply `import usertour from 'usertour.js'` and use it. Example:

```js
import usertour from 'usertour.js'

var options = {
  userInfo: {
    id: 'USER_ID',
    name: 'USER_NAME'
  }
}
usertour.init('ENV_ID', options)
```
